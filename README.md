# 🎵 Music Player Web Application

This is a simple Music Player Web Application created using HTML, CSS and JavaScript.  
The project allows users to play music with basic controls and provides a clean and interactive user interface.

## Features
- Play and Pause music
- Next and Previous controls
- Playlist handling
- Simple and responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## About the Project
This project was built to practice JavaScript concepts like DOM manipulation and audio control while designing a user-friendly web interface.
